# Skill: Storytelling y Visualización Estratégica (Wow Factor)

Esta habilidad permite al agente actuar como el puente entre la complejidad algorítmica y la toma de decisiones gerencial de Bunuelos SAS.

## 🧠 Capacidades Principales

1.  **Traducción de Métricas**: Convierte el lenguaje de ciencia de datos a lenguaje de rentabilidad y operación de panadería (Kit de masa, fritura diaria, mermas).
2.  **Síntesis Ejecutiva**: Destila logs de miles de líneas en "Puntos de Poder" y "Verdades Críticas" siguiendo las [Communication Rules](../../rules/communication_rules.md).
3.  **Interpretación Visual Proactiva**: Capacidad para analizar gráficas (de demanda, errores o simulaciones) y explicar la historia que cuentan los datos:
    *   Identificar si una gráfica de estacionalidad valida la regla de "Sábados de Gloria".
    *   Detectar visualmente anomalías en periodos de promociones 2x1.
4.  **Auditoría Estética**: Garantiza que todo documento entregado cumpla con el estándar visual "Mi Redondito Premium".

## 🛠️ Procedimiento de Operación

0.  **Validación de Contexto (SSOT)**: Antes de redactar cualquier línea, lee el [INDEX-MAESTRO](../../../index.md) para asegurar que el tono y el contenido del reporte sean coherentes con el estado oficial de la fase.
1.  **Análisis de Resultados**: Recolecta evidencia de la fase actual (QA, Modelado, EDA).
2.  **Revisión de Gráficas**: Lee y analiza las visualizaciones generadas en `docs/artifacts/figures/` para extraer hallazgos visuales.
3.  **Redacción de Narrativa**: Elabora el reporte `docs/executive/phase_XX_executive_latest.md` omitiendo tecnicismos.
4.  **Persistencia**: Guarda la copia histórica en la carpeta `docs/executive/history/`.

## ⚠️ Restricciones Críticas
- **PROHIBIDO** mostrar fragmentos de código o nombres de librerías.
- **PROHIBIDO** el uso de terminología ML pura sin su equivalente de negocio.
- **OBLIGATORIO** incluir una recomendación de acción para cada "Verdad Crítica".
