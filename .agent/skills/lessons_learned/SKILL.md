# Habilidad: Lessons Learned Management (PM)

Capacidad para capturar, categorizar y aplicar el conocimiento derivado de la experiencia del proyecto.

## Capacidades

1.  **Reflexión Crítica**: Analizar el historial de conversación y los resultados de las tareas para identificar patrones de éxito o fallo.
2.  **Análisis de Causa Raíz**: Utilizar técnicas como los "5 Porqués" para entender el origen de las desviaciones.
3.  **Documentación Estructurada**: Mantener el `lessons_learned_log.md` con entradas claras y accionables.
4.  **Transferencia de Conocimiento**: Incorporar aprendizajes previos en la planificación de nuevas fases.

## Directrices de Operación

- **Fase de Cierre**: Al recibir una orden de "cerrar fase", el agente debe activar esta habilidad automáticamente.
- **Narrativa Positiva**: Las lecciones deben redactarse como oportunidades de mejora, no como incidentes negativos.
- **Trazabilidad**: Relacionar las lecciones con los IDs de los requerimientos `[REQ]` o tareas `[TSK]` cuando sea posible.
